# Автогенератор сценариев для Reels, Shorts и TikTok

## Проблема

Снимать короткие видео регулярно — сложно. Каждый раз нужно придумывать структуру, хук, текст на экране, закадровый голос и CTA. Это отнимает время даже у опытных авторов, а новичкам — тем более.

## Пользователь

- Контент-мейкеры и блогеры, которые ведут Reels / Shorts / TikTok
- Маркетологи, которые делают видео для продуктов и услуг
- Малый бизнес, которому нужен видеоконтент без найма сценариста

## Основная функция

Пользователь вводит тему, продукт, целевую аудиторию и выбирает стиль видео — сервис генерирует готовый сценарий на 60 секунд через AI-модель.

## Формат результата

На выходе пользователь получает:

- **Заголовок** — цепляющее название видео (до 60 символов)
- **Хук** — первые 3–5 секунд, которые останавливают скролл
- **Сцены** — 4–6 блоков с описанием действия, текстом на экране и закадровым текстом
- **CTA** — конкретный призыв к действию в конце

Готовый сценарий можно скопировать одной кнопкой.

## Минимальный функционал (MVP)

- [x] Форма ввода: тема, продукт/услуга, аудитория, стиль видео
- [x] Генерация сценария через AI (GPT-5 Mini)
- [x] Структурированный вывод по блокам
- [x] Кнопка копирования всего сценария
- [x] Обработка ошибок

## Запуск локально

### Требования

- Python 3.10+
- Node.js 18+
- Ключ ProxyAPI (или OpenAI API)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ksugrukz/reels-script-generator.git
cd reels-script-generator
```

### 2. Создать `.env` в корне

```
PROXYAPI_KEY=your_key_here
```

### 3. Запустить бэкенд

```bash
pip install -r requirements.txt
python -m uvicorn backend.main:app --reload --port 8001
```

### 4. Запустить фронтенд

```bash
cd frontend
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000)
